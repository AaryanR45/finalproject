The Project

What Is PowerPlay?

PowerPlay is a full-stack sports e-commerce platform. It features a content-based product recommendation engine built entirely from scratch using TF-IDF and Cosine Similarity. When you view or search a product, the system finds similar items based on their description tags. Payments are handled through eSewa, Nepal's leading digital wallet.

Recommendations

TF-IDF + Cosine Similarity built from scratch.

Admin Panel

Full dashboard to manage products, users, categories and view all orders.

MySQL Database

Relational data model with users, products, carts, and purchase records.
